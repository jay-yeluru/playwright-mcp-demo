# Demo Application Plan

This is a test plan for the TodoMVC application demo.

## Authentication
(No authentication needed for TodoMVC)

## Scenarios
We need to test basic Todo functionality.

### 1. Adding a new Todo
- Enter a todo item
- Press Enter
- Verify it is added to the list

### 2. Changing Todo status
- Add a new todo
- Mark it as completed
- Verify it is checked

### 3. Deleting a Todo
- Add a todo
- Delete it
- Verify the list is empty
